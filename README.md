# ping-pong-Game

VsCode için LiverServer eklentisiyle kolay bir şekilde çalıştırabilirsiniz.

### Kurallar

Sol tarafta ki oyuncu "W" ve "S" tuşlarıyla,
Sağ tarafta ki oyuncu "arrow UP" ve "arrow DOWN" tuşlarıyla oynar.
Oyun 5'te biter.
